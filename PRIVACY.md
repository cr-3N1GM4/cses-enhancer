Privacy Policy for CSES Enhancer

CSES Enhancer does not collect, store, or transmit any personal data.

All user data, including notes and starred problems, is stored locally on the user's device using browser storage (chrome.storage.local). This data never leaves the user's device and is not shared with any third parties.

The extension does not use analytics, tracking, or external servers.

Permissions:
- storage: Used only to save user notes and starred problems locally.
